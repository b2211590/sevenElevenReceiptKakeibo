# sevenElevenReceiptKakeibo

## 概要

セブンイレブンのレシートの画像をLINEbotに上げることで, 日時と合計金額を認識してくれるサービスのようなものです.

レシートの文字を読み取るのに AWS の Rekognition-detect_text を使用しています.
